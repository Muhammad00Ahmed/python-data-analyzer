# Python Data Analyzer

Python scripts for data analysis with pandas, visualization with matplotlib, and statistical computations.

## Features

- CSV data processing
- Statistical analysis
- Data visualization
- Data cleaning utilities
- Export reports
- Multiple chart types

## Requirements

```bash
pip install pandas matplotlib numpy seaborn
```

## Usage

```python
python analyze_data.py input.csv
```

## Scripts Included

- `analyze_data.py` - Main analysis script
- `visualize.py` - Data visualization
- `clean_data.py` - Data cleaning utilities
- `statistics.py` - Statistical computations
